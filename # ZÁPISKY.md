## Strip 

- Používá se na vyčištění textu  
```python
Txt = ("AHAhaHAhaHAh") 
Txt = Txt.Strip()
```
## Lower 

- Všechno se dá do malích písmen 
```python
Txt = ("AHAhaHAhaHAh") 
Txt = Txt.lower() 
```
## Split 
- Rozdělí tetxt podle mezer 
```python
Txt = ("AHAhaH  AhaHAh") 
Txt = Txt.split() 
```
## Slovník

slovnik = {
